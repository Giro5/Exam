Задание для экзамена на модуле

Создать класс для работы со структурой данных:
кол-во полей или свойств класса определяется кол-м полей для ввода данных на форме
В классе должны быть методы для реализции программы след функционала:
программа должна обеспечивать работу с заявками на билеты в аэропорту
ввод пользователем и добавление в конец файла новой записи содержащий след данные(поля):
1)пункт назначения (пользователь выбирает вариант из готовых списков городов
2)номер рейса ( тоже выбирается из списка)
3)ФИО пассажира 
4) желаемая дата выезда
Вывод на экран полного перечня записей, ранеее внесенных в файл, и экспорт их в приложение Ворд или Эксель
Удалние из файла по нажатию кнопки Удалить записи с опред-м номером рейса указанным пользователем
Поиск записи\записей по запросу(по номеру рейса или дате) и вывод на экран всех записей отвечающих поисковому запросу(в случае успеха)
В случае если файл выполняющий роль базы данных не найден, программа сообщает пользоватею об этой ситуации и создает новый пустой файл
Действия с данными должны выполняться через Меню или Кнопки
